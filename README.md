# Arachnophobia Mode

A RuneLite plugin designed to make Old School RuneScape more accessible for players with arachnophobia. This plugin obscures spiders, nylocas, and any other user-defined NPCs by rendering them as solid black silhouettes, removing their terrifying textures and details.

## Features

* **Default Protection:** Automatically obscures all standard spiders and all variants of Nylocas (including those in the Theatre of Blood), as well as spider bosses like Sarachnis, Venenatis, Araxxor, and Spindel.
* **Customizable Threat List:** A single, easy-to-use text box in the plugin settings allows you to add or remove any NPC in the game. Don't want to hide Sarachnis? Just delete her name from the list. Terrified of Scabarites? Add them in!
* **Shift-Click Integration:** Hold `Shift` and right-click any NPC in the game to quickly "Obscure" or "Reveal" them on the fly. The plugin will automatically update your settings list in the background.

## How to Use

1. Turn the plugin on in the RuneLite plugin panel.
2. If you encounter an NPC that triggers your phobia that isn't on the default list, hold **Shift**, right-click the NPC, and select **Obscure**.
3. If you want to see an NPC again, hold **Shift**, right-click the black silhouette, and select **Reveal**.
4. To manually edit the list, open the plugin configuration (the gear icon) and type the exact names of the NPCs you want to hide, separated by commas. Hit `Enter` to apply your changes.

## Important Note for PvM

Because this plugin entirely obscures the model of the NPC, it also hides their attack animations. If you are doing high-level PvM content where reading an NPC's attack animation is required (such as Verzik Vitur, Araxxor, or Venenatis), you will have to rely on audio cues or other plugins to know what attack is coming. Use at your own risk during difficult encounters!
